# **One Mile at a Time**



## About & Features


**One Mile at a Time** is a full-stack single-page travel blog site.

### Front-end Technology : 
- **React, React-Router-Dom**
- **React-Bootstrap**
- **CSS3**
- **Firebase Authentication**


### Back-end Technology : 
- **Node**
- **Express**
- **MongoDB**
